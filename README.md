## Descripción General
Scholarly es un dashboard académico de gestión de cursos desarrollado con React 19 + Vite. Permite la gestión completa de cursos y certificaciones, además de permitir mostrar los logros de cada usuario de manera pública.

## Instalación y Ejecución

Instalar dependencias:
```bash
npm install

```

Ejecutar en modo desarrollo:

```bash
npm run dev

```

Verificar ESLint:

```bash
npm run lint

```
